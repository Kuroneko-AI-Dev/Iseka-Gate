npm run dev -- --host #menjalankan frontend network

5. Jalankan Backend FastAPI

Di terminal backend:

uvicorn main:app --host 0.0.0.0 --port 8000 --reload

Kalau berhasil akan muncul:

INFO: Uvicorn running on http://0.0.0.0:8000
6. Jalankan Cloudflare Tunnel

Buka terminal baru:

cloudflared tunnel --url http://localhost:8000

Nanti akan muncul URL seperti:

https://xxxxxxxx.trycloudflare.com

Salin URL tersebut.

Jika URL Cloudflare berubah

Ubah:

VITE_API_URL=https://xxxxxxxx.trycloudflare.com

Kemudian:

cd frontend
npm run build

Lalu upload ulang folder:

frontend/dist

ke Netlify.

ganti juga domain yang berada di firebase
